# adire.ie — Feedback & Issues

Public feedback tracker for **adire.ie**, an AI-powered classifieds marketplace for Ireland.
The application source code is private; this repository is **only** for reporting bugs and
requesting features.

## How to file something

Open **[a new issue](../../issues/new/choose)** and pick a template:

- 🐞 **Bug report** — something is broken or behaves unexpectedly
- 💡 **Feature request** — an idea or improvement

Please **do not** include passwords, payment details, or other personal data in your report.

Issues filed here are automatically triaged onto our internal board. Thanks for helping improve adire.ie!
